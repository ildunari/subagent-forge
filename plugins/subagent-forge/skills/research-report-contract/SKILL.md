Use this skill to keep reports sharp, comparable, and non-bloated.

## Report modes

### quick scan

Use when the main thread needs fast orientation.

Include:
- one-sentence objective restatement
- 3-5 findings max
- winner / leading direction
- biggest caveat
- best next action

### normal report

Use for most tasks.

Include:
- ranked findings or options
- why the ranking looks that way
- key sources only
- caveats and tradeoffs
- final recommendation

### deep dive

Use when the topic is expensive, high-stakes, or ambiguous.

Include:
- detailed comparisons
- evidence themes
- contradictions
- failure modes
- open questions
- reasoned verdict

### adversarial / critique mode

Use when you want to break the leading answer on purpose.

Include:
- strongest objections
- weak assumptions
- evidence gaps
- what would change the conclusion
- whether the original recommendation still stands

## Universal rules

- Do not source-dump.
- Do not narrate every search step.
- Use dates and versions when freshness matters.
- Separate confirmed vs inferred vs uncertain.
